# v0.3.0 — Advanced Integrations, Webhooks & Multi-Account

**Status:** Ideation
**Target:** Future minor release (post v0.2.0)

---

## Theme

Deep integrations, real-time event handling via webhooks/resources, multi-account management, and expanded API coverage.

---

## Features

### 1. MCP Resources (real-time data subscriptions)

Expose frequently accessed data as MCP Resources that clients can subscribe to:

| Package | Planned resources |
|---------|------------------|
| Stripe | Active subscription count, MRR summary, recent failed payments |
| Sentry | Unresolved issue count by project, error rate trends |
| Datadog | Monitor status summary, active alerts |
| PagerDuty | Current on-call schedule, open incident count |
| Vercel | Deployment status by project |

### 2. Webhook ingestion

Accept inbound webhooks to provide real-time context without polling:

- Stripe webhook events (payment succeeded/failed, subscription changes)
- Sentry issue alerts
- PagerDuty incident triggers
- Linear issue updates
- Expose webhook-derived state via MCP Resources

### 3. Multi-account / multi-workspace support

- Connect multiple Stripe accounts, Sentry orgs, or Linear workspaces simultaneously
- Account selection via tool parameter or resource prefix
- Token storage scoped per account in oauth-core

### 4. Expanded API coverage

| Package | New tool areas |

| Stripe | Disputes, coupons, promotion codes, webhook endpoints |
| Sentry | Releases, deploys, performance transactions |
